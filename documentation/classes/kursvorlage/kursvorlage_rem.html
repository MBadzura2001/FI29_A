Kursvorlage::Rem

Kursvorlage::Rem - Entfernt ein Modul Objekt aus einem Kursvorlage Objekt

Beschreibung

public Rem ( object $objekt )

Parameter-Liste

objekt
Das Modul Objekt das entfernt werden soll. Verfügt das zu entfernende Modul Objekt über einen Wert für Unterrichtseinheiten, wird dieser von den Unterrichtseinheiten des Modul Objekt, aus dem es entfernt wird, abgezogen.
Der Array dem dieses Modul Objekt zugewiesen wurde, reduziert sich dabei in seiner größe.

Fehler / Ausnahmen

ObjectSelfAdditionException
Wird ausgelöst, wenn das zu entfernende Objekt durch die selbe Klasse instanziiert wurde.
InvalidObjectAdditionException
Wird ausgelöst, wenn das zu entfernende Objekt nicht durch die Klasse 'Modul' instanziiert wurde.
ObjectAllreadyInCollectionException
Wird ausgelöst, wenn das zu entfernende Objekt nicht in der Eigenschaft data des Kursvorlage Objekt enthalten ist.

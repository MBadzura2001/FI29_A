Lernfeld::__construct

Lernfeld::__construct - Erstellt eine Lernfeld instanz

Beschreibung

public __construct ( string $bezeichnung , int $ue = 0 )

Parameter-Liste

bezeichnung
Die Bezeichnung die dieses Objekt tragen soll.
ue
Die Anzahl der Unterrichtseinheiten dieses Objektes. Diese Angabe ist für Lernfeld Objekte zwingend Erforderlich, da sich die Dauer von Modul Objekten aus der Summe ihrer Lernfeld Objekte ergibt. Aus der Dauer von Modul Objekten ergibt sich somit letztendlich auch die Gesamtdauer von Kursvorlage Objekten.

Rückgabewerte

Erstellt ein Lernfeld Objekt bei Erfolg

Beispiel(e)

< ?php
 /* Lernfeld Objekt erstellen */
 $lf1 = new Lernfeld("Beispiel Lernfeld", 80);

echo $lf1->bezeichnung; . ', ' . $lf1->ue
/* Gibt 'Beispiel Lernfeld, 80' aus */
 ?>

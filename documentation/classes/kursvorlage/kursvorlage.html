Kursvorlage Klasse

Repräsentiert Kursvorlage Objekte für Kursvorlagen.

Diese Klasse spezialisiert die KursObjekt Klasse

Klassenbeschreibung

Kursvorlage{
public __construct ( string $bezeichnung )
public Add ( object $objekt )
public Rem ( object $objekt )
public Update ( void )
}

Eigenschaften

string bezeichnung
Variable zum Speichern einer Bezeichnung für das Kursvorlage Objekt
int ue
Variable zum Speichern der Anzahl an Unterrichtseinheiten für das Kursvorlage Objekt
array data
Variable zum Hinzufügen von Modul Objekten zum Kursvorlage Objekt

Inhaltsverzeichnis

  • Kursvorlage::__construct - Erstellt ein Kursvorlage Objekt bei Erfolg.
  • Kursvorlage::Add - Fügt ein Modul Objekt hinzu.
  • Kursvorlage::Rem - Entfernt ein Modul Objekt.
  • Kursvorlage::Update - Löst eine Aktualisierung der $ue Eigenschaft eines Kursvorlage Objektes aus.

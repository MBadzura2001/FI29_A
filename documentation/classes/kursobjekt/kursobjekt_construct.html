KursObjekt::__construct

KursObjekt::__construct - Erstellt eine KursObjekt instanz

Beschreibung

public __construct ( string $bezeichnung [, int $ue = 0] )

Parameter-Liste

bezeichnung
Die Bezeichnung die dieses Objekt tragen soll.
ue
Die Anzahl der Unterrichtseinheiten dieses Objektes. Optional, beträgt bei auslassen 0.

Rückgabewerte

Erstellt ein KursObjekt Objekt bei Erfolg

Beispiel(e)

< ?php
 /* KursObjekt mit Bezeichnung, ohne Unterrichtseinheiten erstellen */
 $ko1 = new KursObjekt("Beispiel KursObjekt");

echo $ko1->bezeichnung;
echo $ko1->ue;
/* Gibt 'Beispiel KursObjekt' und 0 aus */

/*KursObjekt mit Bezeichnung und Unterrichtseinheiten erstellen */
$ko2 = new KursObjekt("Beispiel KursObjekt", 80);

echo $ko2->bezeichnung;
echo $ko2->ue;
/* Gibt 'Beispiel KursObjekt' und 80 aus */
 ?>

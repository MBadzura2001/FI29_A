Lernfeld Klasse

Repräsentiert Lernfeld Objekte für Kursvorlagen.

Diese Klasse spezialisiert die KursObjekt Klasse

Klassenbeschreibung

Lernfeld{
public __construct ( string $bezeichnung )
public Add ( object $objekt )
public Rem ( object $objekt )
public Update ( void )
}

Eigenschaften

string bezeichnung
Variable zum Speichern einer Bezeichnung für das Lernfeld Objekt
int ue
Variable zum Speichern der Anzahl an Unterrichtseinheiten für das Lernfeld Objekt
array data
Variable zum Hinzufügen von KursObjekt Objekten zum Lernfeld Objekt

Inhaltsverzeichnis

  • Lernfeld::__construct - Erstellt ein Lernfeld Objekt bei Erfolg.
  • Lernfeld::Add - Fügt ein Schwerpunkt Objekt hinzu.
  • Lernfeld::Rem - Entfernt ein Schwerpunkt Objekt.
  • Lernfeld::Update - Löst eine Ausnahme aus, da Objekte dieses Typs keine Objekte mit der Eigenschaft $ue aufnehmen.

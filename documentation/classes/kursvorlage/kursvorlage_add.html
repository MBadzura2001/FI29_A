Kursvorlage::Add

Kursvorlage::Add - Fügt einem Kursvorlage Objekt ein Modul Objekt hinzu

Beschreibung

public Add ( object $objekt )

Parameter-Liste

objekt
Das Modul Objekt das hinzugefügt werden soll. Verfügt das hinzuzufügende Modul Objekt über einen Wert für Unterrichtseinheiten, wird dieser zu den Unterrichtseinheiten des Kursvorlage Objekt, dem es hinzugefügt wird, angerechnet.

Fehler / Ausnahmen

ObjectSelfAdditionException
Wird ausgelöst, wenn das hinzuzufügende Objekt durch die selbe Klasse instanziiert wurde.
InvalidObjectAdditionException
Wird ausgelöst, wenn das hinzuzufügende Objekt nicht durch die Klasse 'Modul' instanziiert wurde.
ObjectAllreadyInCollectionException
Wird ausgelöst, wenn das hinzuzufügende Objekt bereits in der Eigenschaft data des Kursvorlage Objekt enthalten ist.

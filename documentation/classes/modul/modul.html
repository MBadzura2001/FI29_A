Modul Klasse

Repräsentiert Modul Objekte für Kursvorlagen.

Diese Klasse spezialisiert die KursObjekt Klasse

Klassenbeschreibung

Modul{
public __construct ( string $bezeichnung )
public Add ( object $objekt )
public Rem ( object $objekt )
public Update ( void )
}

Eigenschaften

string bezeichnung
Variable zum Speichern einer Bezeichnung für das Modul Objekt
int ue
Variable zum Speichern der Anzahl an Unterrichtseinheiten für das Modul Objekt
array data
Variable zum Hinzufügen von KursObjekt Objekten zum Modul Objekt

Inhaltsverzeichnis

  • Modul::__construct - Erstellt ein Modul Objekt bei Erfolg.
  • Modul::Add - Fügt ein Lernfeld Objekt hinzu.
  • Modul::Rem - Entfernt ein Lernfeld Objekt.
  • Modul::Update - Löst eine Aktualisierung der $ue Eigenschaft eines Modul Objektes aus.

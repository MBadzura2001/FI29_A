Kursvorlage::__construct

Kursvorlage::__construct - Erstellt eine Kursvorlage instanz

Beschreibung

public __construct ( string $bezeichnung [, int $ue = 0] )

Parameter-Liste

bezeichnung
Die Bezeichnung die dieses Objekt tragen soll.
ue
Die Anzahl der Unterrichtseinheiten dieses Objektes. Optional, beträgt bei auslassen 0.

Rückgabewerte

Erstellt ein Kursvorlage Objekt bei Erfolg

< ?php
 /* Kursvorlage Objekt erstellen */

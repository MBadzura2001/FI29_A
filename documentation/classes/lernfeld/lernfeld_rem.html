Lernfeld::Rem

Lernfeld::Rem - Entfernt ein Lernfeld Objekt aus einem Lernfeld Objekt

Beschreibung

public Rem ( object $objekt )

Parameter-Liste

objekt
Das Schwerpunkt Objekt das entfernt werden soll.

Fehler / Ausnahmen

ObjectSelfAdditionException
Wird ausgelöst, wenn das zu entfernende Objekt durch die selbe Klasse instanziiert wurde.
InvalidObjectAdditionException
Wird ausgelöst, wenn das zu entfernende Objekt nicht durch die Klasse 'Schwerpunkt' instanziiert wurde.
ObjectAllreadyInCollectionException
Wird ausgelöst, wenn das zu entfernende Objekt nicht in der Eigenschaft data des Lernfeld Objekt enthalten ist.

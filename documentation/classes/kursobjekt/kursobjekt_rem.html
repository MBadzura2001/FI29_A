KursObjekt::Rem

KursObjekt::Rem - Entfernt ein KursObjekt Objekt aus einem anderen KursObjekt Objekt

Beschreibung

public Rem ( object $objekt )

Parameter-Liste

objekt
Das KursObjekt Objekt das entfernt werden soll. Verfügt das zu entfernende KursObjekt Objekt über einen Wert für Unterrichtseinheiten, wird dieser von den Unterrichtseinheiten des KursObjekt Objekt, aus dem es entfernt wird, abgezogen.
Der Array dem dieses KursObjekt Objekt zugewiesen wurde, reduziert sich dabei in seiner größe.

Beispiel(e)

< ?php
 /* KursObjekt mit Bezeichnung, ohne Unterrichtseinheiten erstellen */
 $ko1 = new KursObjekt("Modultest");

/*KursObjekt mit Bezeichnung und Unterrichtseinheiten erstellen */
$ko2 = new KursObjekt("Lernfeld Versuch", 80);
$ko3 = new KursObjekt("Lernfeld die Zweite", 80);

/* Erstem KursObjekt $ko1 die anderen KursObjekt Objekte $ko2 und $ko3 hinzufügen */
$ko1->Add($ko2);
$ko1->Add($ko3);

?>
< pre>
< ?= print_r($ko1) ?>
< /pre>
/* Erzeugt folgende Ausgabe: */


$ko1->Rem($ko2);

?>
< pre>
< ?= print_r($ko1) ?>
< /pre>
/* Erzeugt folgende Ausgabe: */

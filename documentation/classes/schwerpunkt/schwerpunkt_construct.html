Schwerpunkt::__construct

Schwerpunkt::__construct - Erstellt eine Schwerpunkt instanz

Beschreibung

public __construct ( string $bezeichnung )

Parameter-Liste

bezeichnung
Die Bezeichnung die dieses Objekt tragen soll.

Rückgabewerte

Erstellt ein Schwerpunkt Objekt bei Erfolg

Beispiel(e)

< ?php
 /* Schwerpunkt Objekt instanziieren und die Eigenschaft 'bezeichnung' auslesen */
 $sp1 = new Schwerpunkt("Beispiel Schwerpunkt");

echo $sp1->bezeichnung;
/* Gibt 'Beispiel Schwerpunkt' aus */
 ?>

KursObjekt::Update

KursObjekt::Update - Sorgt fur eine Neuermittlung der anzahl an Unterrichtseinheiten eines KursObjekt Objekt durch summieren der Unterrichtseinheiten aller KursObjekt Objekte die es enthält.

Beschreibung

public Update ()

Beispiel(e)

< ?php
 /* KursObjekt mit Bezeichnung, ohne Unterrichtseinheiten erstellen */
 $ko1 = new KursObjekt("Modultest");

/*KursObjekt mit Bezeichnung und Unterrichtseinheiten erstellen */
$ko2 = new KursObjekt("Lernfeld Versuch", 80);
$ko3 = new KursObjekt("Lernfeld die Zweite", 80);

/* Erstem KursObjekt $ko1 das KursObjekt $ko2 hinzufügen - die Dauer vom Kursobjekt $ko1 sollte sich korrekt erhöhen. */
$ko1->Add($ko2);
$ko2->Add($ko3);

/* Zweitem KursObjekt $ko2 das KursObjekt $ko3 hinzufügen - die Dauer vom KursObjekt $ko2 sollte sich korrekt erhöhen, jedoch besitzt das KursObjekt $ko1 darüber keine kenntnis. Damit sich seine dauer korrekt erhöht, stoßen wir das Update dafür manuell an. */
$ko1->Update();

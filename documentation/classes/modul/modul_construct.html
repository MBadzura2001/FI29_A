Modul::__construct

Modul::__construct - Erstellt eine Modul instanz

Beschreibung

public __construct ( string $bezeichnung [, int $ue = 0] )

Parameter-Liste

bezeichnung
Die Bezeichnung die dieses Objekt tragen soll.
ue
Die Anzahl der Unterrichtseinheiten dieses Objektes. Optional, beträgt bei auslassen 0.

Rückgabewerte

Erstellt ein Modul Objekt bei Erfolg

Beispiel(e)

< ?php
 /* Modul Objekt erstellen */
 $mo1 = new Modul("Beispiel Modul");

echo $mo1->bezeichnung;
/* Gibt 'Beispiel Modul' aus*/
?>

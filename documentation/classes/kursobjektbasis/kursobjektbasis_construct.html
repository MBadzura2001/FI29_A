KursObjektBasis::__construct

KursObjektBasis::__construct - Erstellt eine KursObjektBasis instanz

Beschreibung

public __construct ( string $bezeichnung )

Parameter-Liste

bezeichnung
Die Bezeichnung die dieses Objekt tragen soll.

Rückgabewerte

Erstellt ein KursObjektBasis Objekt bei Erfolg

Beispiel(e)

< ?php
 /* KursObjektBasis Objekt instanziieren und die Eigenschaft 'bezeichnung' auslesen */
 $kob = new KursObjektBasis("Beispielobjekt");

echo $kob->bezeichnung;
/* Gibt 'Beispielobjekt' aus */
 ?>

KursObjektBasis Klasse

Repräsentiert Objekte für Kursvorlagen welche über eine Bezeichnung verfügen müssen.

Diese Klasse bildet die Grundlage für die Klassen KursObjekt, TrainerObjekt, Schwerpunkt

Klassenbeschreibung

KursObjektBasis{
public __construct ( string $bezeichnung )
}

Eigenschaften

string bezeichnung
Variable zum Speichern einer Bezeichnung für das KursObjektBasis Objekt

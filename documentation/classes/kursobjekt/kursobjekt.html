KursObjekt Klasse

Repräsentiert Objekte für Kursvorlagen.

Diese Klasse spezialisiert die KursObjektBasis Klasse

Klassenbeschreibung

KursObjekt{
public __construct ( string $bezeichnung )
public Add ( object $objekt )
public Rem ( object $objekt )
public Update ( void )
}

Inhaltsverzeichnis

  • KursObjekt::__construct - Erstellt ein KursObjekt Objekt bei Erfolg.
  • KursObjekt::Add - Fügt ein KursObjekt Objekt hinzu.
  • KursObjekt::Rem - Entfernt ein KursObjekt Objekt.
  • KursObjekt::Update - Löst eine Aktualisierung der $ue Eigenschaft eines KursObjekt Objektes aus.

KursObjekt::Add

KursObjekt::Add - Fügt einem KursObjekt ein weiteres KursObjekt hinzu

Beschreibung

public Add ( object $objekt )

Parameter-Liste

objekt
Das KursObjekt Objekt das hinzugefügt werden soll. Verfügt das hinzuzufügende KursObjekt Objekt über einen Wert für Unterrichtseinheiten, wird dieser zu den Unterrichtseinheiten des KursObjekt Objekt, dem es hinzugefügt wird, angerechnet.

Beispiel(e)

< ?php
 /* KursObjekt mit Bezeichnung, ohne Unterrichtseinheiten erstellen */
 $ko1 = new KursObjekt("Beispiel KursObjekt");

echo $ko1->bezeichnung;
echo $ko1->ue;
/* Gibt 'Beispiel KursObjekt' und 0 aus */

/*KursObjekt mit Bezeichnung und Unterrichtseinheiten erstellen */
$ko2 = new KursObjekt("Test KursObjekt mit Unterrichtseinheiten", 80);

echo $ko2->bezeichnung;
echo $ko2->ue;
/* Gibt 'Test KursObjekt mit Unterrichtseinheiten' und 80 aus */

/* Erstem KursObjekt $ko1 das zweite KursObjekt $ko2 hinzufügen */
$ko1->Add($ko2);

?>
< pre>
< ?= print_r($ko1) ?>
< /pre>
/* Erzeugt folgende Ausgabe: */

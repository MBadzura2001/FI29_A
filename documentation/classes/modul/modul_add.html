Modul::Add

Modul::Add - Fügt einem modul Objekt ein Lernfeld Objekt hinzu

Beschreibung

public Add ( object $objekt )

Parameter-Liste

objekt
Das Lernfeld Objekt das hinzugefügt werden soll. Verfügt das hinzuzufügende Lerfeld Objekt über einen Wert für Unterrichtseinheiten, wird dieser zu den Unterrichtseinheiten des Modul Objekt, dem es hinzugefügt wird, angerechnet.

Fehler / Ausnahmen

ObjectSelfAdditionException
Wird ausgelöst, wenn das hinzuzufügende Objekt durch die selbe Klasse instanziiert wurde.
InvalidObjectAdditionException
Wird ausgelöst, wenn das hinzuzufügende Objekt nicht durch die Klasse 'Lernfeld' instanziiert wurde.
ObjectAllreadyInCollectionException
Wird ausgelöst, wenn das hinzuzufügende Objekt bereits in der Eigenschaft data des Modul Objekt enthalten ist.
